实现功能

  • List模块

    增删

    1. 创建计划
    2. 复制计划
    3. 删除计划

    筛选

    1. 手动输入名称模糊查询
    2. 仅我创建
    3. 计划类型
    4. 计划状态
    5. 触达通道

    排序

    1. 触达人次/完成率
    2. 累计完成率

  • Board模块

    增删

    1. 创建计划新增计划
    2. 新增分类
    3. 编辑分类名称
    4. 删除分类

    筛选

    1. 手动输入名称模糊查询

  • Editor模块

    创建

    1. 定时型-单次
    2. 定时型-重复
    3. 触发性-完成A
    4. 触发性-完成A未完成B

    计划定义

    1. 设置计划名称
    2. 新建分组和加入分组

    受众用户设置

    1. 客户群--全部用户或部分用户
    2. 部分用户支持用户属性及用户行为匹配
    3. 预估客户群人数

    触发条件设置定时型

    1. 计划类型--单次或重复
    2. 触发时间--单次型设置具体时间点，重复型设置时间段内每天时间点
    3. 对照组--添加对照组，百分比设置范围为1%至100%

    触发条件设置触发型

    1. 计划类型--完成事件A或者完成事件A后，一段事件内未完成事件B
    2. 触达时间--支持完成事件后立即触达或延时触达
    3. 起止事件--配置时间段
    4. 对照组--添加对照组，百分比设置范围为1%至100%

    触发方式设置

    1. 触达通道
      1. 推送(极光推送，个推推送)--支持插入用户属性，点击后打开可选应用，URL或透传
      2. 短信(极光短信)--设置模板ID，内容变量，测试发送
      3. webhook--支持设置静态动态参数
    2. 勿扰设置--时间段内触发不推送
    3. 全局触达设置--是否使用全局触达设置

    目标设置

    1. 首要目标，用户在n分钟内完成设置的时间，认为完成目标
    2. 支持设置次要目标

  • Report

    未审批计划可查看计划设置，暂停，审批，删除该计划(根据权限判断可执行的操作)

    审批通过的计划

    1. table展示计划执行效果
    2. 分析--制表展示计划执行效果
    3. 详情--Editor计划的详情

    短信

    2. 极光短信设置

    1. App push设置
    2. 短信设置
